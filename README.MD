### ESPECIFICAÇÕES

Todas as expressões que possuem operadores (\*,/,-,+) precisam estar entre parênteses para funcionar corretamente. O programa não lida com strings, for e if não utilizam chaves para definir seu escopo, não é utilizado (;) para declarar final de instrução e o programa aceita somente inteiros, conforme acordado com o professor em aula.
